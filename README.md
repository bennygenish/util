Android Libraries
===






Usage
---

Add it to your root `build.gradle` with:
```gradle
allprojects {
    repositories {
        maven { url "https://jitpack.io" }
    }
}
```

Add it to your app `build.gradle` with the correct module and the latest tag version:
```gradle
dependencies {
    compile 'com.github.bennygenish.util:{module-name}:{tag}'
}
```
